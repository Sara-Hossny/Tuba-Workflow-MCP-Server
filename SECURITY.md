# **Security Policy**

## **Reporting a Vulnerability**

If you discover a security vulnerability, we kindly request responsible disclosure. **Please do not open public GitHub issues** related to security.

Instead, please use the **private vulnerability reporting feature** provided by GitHub.

1. Navigate to the main page of this repository.
2. Go to the **"Security"** tab.
3. Click on **"Report a Vulnerability"** to submit your report privately.

Please provide a detailed description of the vulnerability, steps to reproduce it, and any known impact. We will make every effort to acknowledge your report promptly and address the issue.

## **Security Practices**

This repository is configured with automated security tools to identify potential vulnerabilities:

- **Dependabot:** GitHub's Dependabot is enabled to scan dependencies and alert on any known vulnerabilities.
